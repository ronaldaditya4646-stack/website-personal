ABOUT PROJECT

Sistem ini dirancang untuk mempermudah pemantauan kondisi lingkungan secara real-time. Menggunakan mikrokontroler ESP8266 sebagai otak utama yang terhubung dengan sensor suhu, kelembapan, dan tanah. Data dikirimkan langsung ke server ini untuk divisualisasikan, sekaligus memberikan kontrol penuh terhadap perangkat elektronik melalui Relay.

HARDWARE SPECIFICATIONS

NodeMCU ESP8266

Otak utama sistem dengan modul WiFi yang mana sangat berperan aktif dalam mengupdate data pada dashboard dan personal website kali ini.

Sensor DHT11

Sensor suhu dan kelembapan udara yang berperan memantau kondisi pada ruangan indoor dengan output digital yang stabil.

Soil Moisture

Sensor kelembapan tanah kapasitif berperan memantau kondisi dari tanah pada taman halaman rumah untuk otomatisasi penyiraman.

Relay 4 Channel

Saklar elektronik untuk mengontrol arus AC karena output dari ESP8266 sendiri masih dalam bentuk arus DC dan nilai tegangannya pun hanya 3V.
